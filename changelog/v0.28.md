# MISTER v0.28.0 - Single League Mode

> Released: 2026-05-17

Introduces a second game mode — **Single League** — alongside the existing Pyramid mode.

---

## New Features

### Single League Mode

- **Mode selection at game start:** after clicking New Game, players now choose between two modes before setting up their manager.
- **Pyramid mode** — unchanged. Start at the bottom of the ninth division and climb through all leagues worldwide.
- **Single League mode** — choose any one league (from all available leagues, grouped by country). The entire game world is scoped to that league only: clubs, players, finances, fixtures, and promotion/relegation all operate within it.
- **Free club choice:** in Single League mode the player can pick any club from any division in the selected league — no forced starting position.
- **No Cup in Single League mode:** the Cup competition was designed for the multi-league A/B bracket structure and is not available in Single League. The Cup nav item is hidden when playing this mode.
- **Promotion/relegation stays within the league:** season rollover correctly keeps clubs inside the chosen league — no cross-league boundary moves occur.
- **Existing saves unaffected:** saves created before this version default to Pyramid mode transparently on load.

## Technical

- Added `GameMode` type (`'pyramid' | 'single-league'`) and `selectedLeagueId` field to `GameState`.
- `NEW_GAME` action/reducer filters `leagues`, `divisions`, `clubs`, and `players` to the selected league when mode is `single-league`.
- `initSeason` skips `buildCupDraw` in Single League mode.
- NavBar conditionally hides the Cup item based on `gameMode`.
- Localized mode descriptions, league picker, and all new strings in English, Spanish, and Brazilian Portuguese.
