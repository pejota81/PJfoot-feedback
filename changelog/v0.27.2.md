# MISTER v0.27.2 - Fixture Streak Regression Fix

> Released: 2026-05-17

Patch release for a calendar-generation regression detected in fresh saves.

---

## Fixes

- **Home/away streak cap reliability:** fixed cases where fresh seasons could still produce 3 consecutive home/away games in some divisions.

## Technical

- Updated `src/engine/fixtures.ts`:
  - Increased the seeded first-leg round-order search budget to `max(10_000, roundsPerLeg * 300)`.
  - Stops search as soon as a schedule with max venue streak `<= 2` is found.
  - Preserves strict first-leg-before-rematch sequencing introduced in `0.27.1`.
