# MISTER v0.28.3 - Single League naming + manager profile simplification

> Released: 2026-05-17

Patch release focused on consistency in Single League naming and removal of unnecessary manager personal data fields.

---

## Improvements

- **Single League names are now consistent everywhere:** league labels now use each competition's original name across setup and gameplay surfaces (Dashboard, Match Day, Table, Tactics, Season End, Manager Market, and season-end competition news).
- **Manager DOB removed from user flows:** New Game and Account Settings no longer collect or save manager date of birth.
- **Manager age removed from UI:** manager age display has been removed from Manager Market and related manager data paths.
- **Profile/API cleanup:** profile payloads and defaults no longer expose or initialize `managerDateOfBirth`.
